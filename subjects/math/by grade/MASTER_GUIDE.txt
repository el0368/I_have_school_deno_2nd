================================================================================
HOW TO USE THIS GUIDE
================================================================================

Each topic follows this format:
  Topic Name
  LEARN: [Video/Article description]
  PRACTICE: [Exercise description]

LEARN components include:
  • Instructional videos (5-15 minutes)
  • Articles with examples and explanations
  • Visual demonstrations and animations

PRACTICE components include:
  • Interactive exercises (typically 7 questions for mastery)
  • Immediate feedback
  • Hints available
  • Multiple attempts allowed
  • Progress tracking

LEARNING PATH
================================================================================

Typical progression through each lesson:
1. Watch LEARN video (5-15 min)
2. Read article if available (optional)
3. Complete PRACTICE exercises (10-20 min)
4. Achieve mastery (typically 5/5 or 7/7 correct)
5. Move to next lesson

Mastery levels:
- Not started: Haven't attempted
- Attempted: Started but not mastered
- Familiar: Some correct answers
- Proficient: Mostly correct
- Mastered: Demonstrated full understanding
